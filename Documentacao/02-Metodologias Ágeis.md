## Gerenciamento de Projeto


# Metodologia

O projeto adota uma abordagem ágil baseada nos princípios do Scrum, utilizando Kanban para gestão visual das tarefas por meio da ferramenta Trello(pouco usado). O desenvolvimento do firmware é realizado na Arduino IDE, com controle de versão e colaboração assegurados pelo uso do Git/GitHub. Essa metodologia permite organização eficiente, ciclos curtos de entrega e acompanhamento contínuo do progresso.

 

### Ferramentas

As ferramentas utilizadas no projeto foram:

Arduino IDE, nRF Connect, Serial Bluetooth Terminal, GitHub, ChatGPT, Flutter.
